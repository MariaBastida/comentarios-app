# Base de Datos de Comentarios

Este repositorio contiene el esquema de la base de datos para la aplicación de comentarios.

## Cómo usar

1. Clona el repositorio.
2. Ejecuta el script `schema.sql` en tu servidor PostgreSQL para crear las tablas necesarias.

### Comandos para ejecutar el script

```bash
psql -U tu_usuario -d tu_base_de_datos -f schema.sql
